hostid
===

用来打印当前主机的数字化标识

## 补充说明

**hostid命令** 用于打印当前主机的十六进制数字标识。是主机的唯一标识，是被用来限时软件的使用权限，不可改变。

### 语法  

```shell
hostid
```

### 选项  

```shell
--help：显示帮助信息；
--version：显示版本信息。
```

### 实例  

```shell
[root@localhost ~]# hostid
007f0100
```
