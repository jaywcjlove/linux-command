arch
===

显示当前主机的硬件架构类型

## 补充说明

**arch命令** 用于显示当前主机的硬件架构类型。arch命令等同于`命令name -m`在当前的Linux系统下,arch命令输出结果有:i386、i486、i586、alpha、sparc、arm、m68k、mips、ppc、i686等

### 语法  

```shell
arch [选项]...
```

### 选项

```shell
--help 显示此帮助信息并退出
--version 显示版本信息并退出
```

### 实例  

```shell
arch
x86_64
```
