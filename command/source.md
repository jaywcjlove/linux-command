source
===

在当前Shell环境中从指定文件读取和执行命令，命令返回退出状态。

## 补充说明

**source命令** 在当前Shell环境中从指定文件读取和执行命令，命令返回退出状态。

### 语法  

```shell
source [文件名] [参数]
```

### 例子

读取和执行/root/.bash_profile文件。

```shell
[root@localhost ~]# source ~/.bash_profile
```
